Windows Forms Image Gallery Application

This project is a simple image gallery application developed using C# Windows Forms.
It allows users to load images from their computer, display them in a gallery view, and interact with them easily.

🚀 Features
Load multiple images from local storage
Display images in a scrollable gallery (FlowLayoutPanel)
Click to preview selected image in a larger view
Delete selected images from the gallery
Simple and user-friendly interface
🛠 Technologies Used
C# (.NET Framework)
Windows Forms
Visual Studio
📌 How It Works

Users select images using a file dialog, and each image is added dynamically as a thumbnail. Clicking on a thumbnail displays it in a larger preview area. Users can also remove selected images from the gallery.

🎯 Purpose

This project was created for learning purposes to practice:

Windows Forms UI design
Event handling (Click events)
Dynamic control creation
Basic image handling in C#
